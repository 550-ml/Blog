# 指尖飞舞：VSCode / Cursor Vim 快捷键速查

> 目标：只保留高频操作，便于速查。默认处于 Normal 模式（按 `Esc` 回到 Normal）。

## 0. 我最常用 10 条（置顶）

- 全局搜索替换：`:%s/old/new/g`
- 局部替换（当前行）：`:s/old/new/g`
- 回到上一次修改位置：`` `` ``（按两次反引号）
- 在行尾快速追加（如分号）：`A;`
- 到文件开头插入一行：`gg` -> `O`
- 修改 `if` 条件里的词：`ciw`（按词改）
- 复制全文：`ggVGy`
- 批量改变量名：`*` -> `ciw` 改第一个 -> `n` + `.`
- 插入模式临时执行一个 Normal 命令：`Ctrl-o`
- 常见拼写修正：`:s/touch/toch/g`（或反过来）

## 1. 快速移动（导航）

### 行与页面

- 到第一行：`gg`
- 到最后一行：`G`
- 到第 n 行：`nG` 或 `:n`
- 向下/上移动 n 行：`nj` / `nk`
- 向下/上翻页：`Ctrl-f` / `Ctrl-b`
- 下/上滚一行（视窗）：`Ctrl-e` / `Ctrl-y`

### 行内移动

- 行首：`0`
- 行首非空字符：`^`
- 行尾：`$`
- 下一个单词开头：`w`
- 当前/下一个单词结尾：`e`
- 上一个单词开头：`b`
- 上一个单词结尾：`ge`
- 当前行找字符：`f{char}`
- 匹配括号：`%`

### 位置跳转与返回

- 跳转到定义（局部）：`gd`
- 设置标记：`ma`
- 跳回标记：`` `a ``
- 回到更早的跳转位置：`Ctrl-o`
- 回到更新的跳转位置：`Ctrl-i`

## 2. 搜索与定位

- 向后搜索：`/pattern`
- 向前搜索：`?pattern`
- 下一个匹配：`n`
- 上一个匹配：`N`
- 光标词向后搜索：`*`
- 光标词向前搜索：`#`

## 3. 插入与编辑

### 进入插入模式

- 光标前插入：`i`
- 光标后插入：`a`
- 行首插入：`I`
- 行尾插入：`A`
- 下方新起一行：`o`
- 上方新起一行：`O`
- 插入模式临时执行一个 Normal 命令：`Ctrl-o`

### 删除 / 修改

- 删当前字符 / 前一个字符：`x` / `X`
- 删一个单词：`dw`
- 删到行尾：`D`
- 删整行：`dd`
- 改一个字符：`r`
- 连续替换模式：`R`
- 删除并进入插入（单字符）：`s`
- 删除整行并进入插入：`cc` 或 `S`
- 改到行尾并进入插入：`C`
- 词内修改：`ciw`
- 删除括号内内容：`di(`
- 删除包含括号：`da(`
- 切换大小写：`~`

## 4. 复制、粘贴、缩进

- 复制一个单词：`yw`
- 复制到行尾：`y$`
- 复制整行：`yy`
- 删除并进入寄存器：`dd`
- 在后粘贴 / 前粘贴：`p` / `P`
- 右缩进 / 左缩进：`>` / `<`
- 当前行置顶 / 居中 / 置底：`zt` / `zz` / `zb`

## 5. 可视模式（批量处理）

- 字符可视模式：`v`
- 行可视模式：`V`
- 块可视模式：`Ctrl-v`
- 常见流程：`V` 选中多行 -> `j/k` 扩展 -> `y` 复制 -> `p` 粘贴

## 6. 重复、撤销、重做

- 重复上次改动：`.`
- 重复上次查找方向：`;`
- 反向重复查找：`,`
- 撤销：`u`
- 重做：`Ctrl-r`
- 重复 n 次动作：`n{action}`（如 `5j`、`3dd`）

## 7. 命令行模式（`:`）

- 保存：`:w`
- 另存为：`:w filename`
- 退出：`:q`
- 强制退出不保存：`:q!`
- 保存并退出：`:wq` 或 `ZZ`
- 当前行替换：`:s/old/new/g`
- 全文替换：`:%s/old/new/g`

## 8. 文本对象（高频）

公式：`操作 + 范围`，例如 `d + iw`、`c + i"`。

- `iw` / `aw`：单词内 / 单词（含空格）
- `i"` / `a"`：引号内 / 含引号
- `i(` / `a(`：括号内 / 含括号
- `ip` / `ap`：段落内 / 段落

常见操作：`d` 删除，`c` 修改，`y` 复制。

## 9. VSCode / Cursor 常用快捷键（补充）

- 侧边栏开关：`Ctrl-b`
- 快速打开文件：`Ctrl-p`
- 命令面板：`Ctrl-Shift-p`
- 聚焦编辑器组 1：`Ctrl-1`
- 打开设置：`Ctrl-,`

## 10. 个人高频组合（从原笔记整理）

- 在文件开头插入后回到原处：`gg` -> `O` -> 编辑 -> `Ctrl-o`
- 批量改变量名：`*` 定位同词 -> `ciw` 改一个 -> `n` + `.`
- 追加分号：`A;`
- 快速跳到指定行：`17gg` 或 `:17`
